## What This Repository Documents

This OSINT repository documents a statistically significant pattern: **high-visibility "friction" events** (document releases, scandals, media crises) and **institutional "compliance" events** (policy shifts, financial moves, regulatory changes) cluster together on predictable calendar windows.

**Core Finding**: r = +0.6196 correlation at 2-week lag, p = 0.0004 (n = 28 paired observations)

**Key Claim**: Multiple actors—domestic agencies, foreign adversaries, financial institutions—respond to the same low-attention calendar windows, creating simultaneous clustering of events that would otherwise draw scrutiny.

## Important Disclaimers

1. **Correlation ≠ Causation**: The repository documents that events cluster together, not that one causes another.

2. **No Conspiracy Claim**: The pattern is emergent—multiple actors exploiting the same environmental signals without requiring coordination.

3. **Verification Levels**: Each claim is marked as ✅ VERIFIED, ⚠️ PARTIALLY VERIFIED, or 🔍 HYPOTHESIS.
