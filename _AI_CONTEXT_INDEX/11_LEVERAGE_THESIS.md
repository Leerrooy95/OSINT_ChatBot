## The Pattern Generalized: Thermostat Model

The leverage thesis extends the Thermostat Model documented in `01_CORE_THEORY.md`:

### Friction Events (lever activation)

- Document releases, investigations, scandals, protests
- Generate public attention and institutional pressure
- Can be deployed selectively via timing
- Cluster around calendar anchors (holidays, FOMC dates, congressional recesses)

### Compliance Events (institutional response)

- Policy shifts, deal announcements, capital deployments, personnel changes
- Follow friction events at measurable lag (7 days median)
- Represent institutional alignment with interests that control leverage

### The Self-Regulating Function

```
If compliance is achieved → friction decreases
If compliance is resisted → friction increases (escalation)
The system self-regulates based on institutional response
```
