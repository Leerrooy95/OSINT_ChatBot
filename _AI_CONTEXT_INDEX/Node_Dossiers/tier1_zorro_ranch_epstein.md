### The 7-Year Gap: A Data Point

The NM investigation was **opened in 2019** after Epstein's death but **immediately closed at the request of federal prosecutors in New York** — before any charges were filed, before the ranch was searched, before victim testimony was formalized. It remained closed for **7 years** (2019–2026).

| Period | Status | Mechanism |
|--------|--------|-----------|
| 2019 | Investigation opened; possible victims interviewed | NM AG office |
| 2019 | Investigation **closed at federal request** | Federal prosecutors (SDNY) requested closure |
| 2019–2026 | 7-year gap — no state-level accountability | Federal preemption of state investigation |
| Jan 30–31, 2026 | DOJ releases 3M+ pages of Epstein files | Forced public accountability |
| Feb 19, 2026 | NM AG **reopens** investigation citing "revelations outlined in the previously sealed FBI files" | Public + congressional pressure |

**Analytical significance**: The 7-year gap (2019–2026) is itself a data point in the regulated friction framework. The 2019 closure — at explicit federal request — halted state accountability immediately after Epstein's death removed the possibility of federal prosecution. The 2026 reopening only occurs after the DOJ file release forces public exposure. Pattern: federal action (closure) → 7-year suppression window → external friction (file release) → compliance (reopening). This mirrors the core framework's r = +0.6196 pattern at state level.
