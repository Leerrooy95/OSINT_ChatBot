## 🔱 The Speaker — Personal OSINT Chatbot

A private, password-protected chat interface powered by a **DigitalOcean AI Agent** with an attached Knowledge Base and live web search.

### How It Works

| Priority | Source | Description |
|----------|--------|-------------|
| 1 | **Knowledge Base** | Files uploaded to your DO Agent's Knowledge Base are searched first |
| 2 | **Training Data** | The model's built-in knowledge is used as a fallback |
| 3 | **Web Search** | If neither source has the answer, live web search is performed |

A system prompt is injected server-side to reinforce this priority order on every request.

### Quick Start (Local)

```bash
# 1. Clone & enter the repo
git clone <this-repo> && cd Personal_Chatbot

# 2. Create a virtual environment
python -m venv .venv && source .venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Configure environment variables
cp .env.example .env
#    → edit .env with your real values

# 5. Run the app
python app.py
```

Open **http://localhost:5000** and log in with your clearance code.

### Deploy on Render

1. Push this repo to GitHub (private recommended).
2. In the [Render Dashboard](https://dashboard.render.com), create a **New Web Service** and connect this repo.
3. Render will auto-detect `render.yaml`. Set the following **Environment Variables** in the dashboard:

| Variable | Description |
|----------|-------------|
| `APP_PASSWORD` | Password to access the chat UI |
| `DO_AGENT_ENDPOINT` | Your DigitalOcean Agent URL ending in `/api/v1` |
| `DO_AGENT_KEY` | Your DigitalOcean Agent API key |
| `FLASK_SECRET_KEY` | Auto-generated by `render.yaml` |

4. Deploy. The health-check at `/health` confirms the service is running.

### Security

- The `DO_AGENT_KEY` is **never** sent to the browser — all API calls go through the Flask backend.
- Session cookies are `HttpOnly` and `SameSite=Lax`.
- The chat UI is gated behind a password login.
- `FLASK_SECRET_KEY` should be set to a long random value in production (auto-generated in `render.yaml`).
